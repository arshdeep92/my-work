This sidebar contains snippets of code to help developers implement the user interface design. Developers can toggle between HTML, CSS/SASS & JavaScript code snippets to see relevant code in each language.

Introduction

The purpose of this styleguide is too enable better collaboration between designers and developers working on the Bel’s website. Treat this document as a user interface specification / documentation / development guide. Good knowledge of user interface design as well as front-end development tools, methods and best practices is beneficial when reading this document.

This is a living document and will evolve as time goes on. If you are working on Bel’s website please check back often.
